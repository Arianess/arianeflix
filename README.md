## ARIANEFLIX

Projeto desenvolvido durante a Imersão React promovida pela Alura. Aplicação inspirada na Nextflix.

## Tecnologias, Ferramentas e Recursos

* HTML

* CSS

* JavaScript

* React

* Node

* npm

* VSCode

* Git

* Vercel

* Heroku

## Funcionalidades

* Cadastrar vídeo.

* Cadastrar categoria.
